Quick Start Guide

Get up and running in minutes with these simple steps 
git clone https://github.com/Franklyc/Transcript-companion.git
cd Transcript-companion
pip install -r requirements.txt

Configuration Steps

  1. Rename configuration files: config.py.example to config.py and prefix.py.example to prefix.py
  2. Set up API keys: Open config.py and replace the placeholders with your actual API keys
  3. Configure default folder: Edit DEFAULT_FOLDER_PATH in config.py to point to your transcript files directory
  4. Customize prefix: Modify the get_original_prefix() function in prefix.py to define initial instructions for the LLM
  5. Launch the application: Run python main.py to start Transcript Companion
